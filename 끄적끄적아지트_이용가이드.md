## 교사 회원가입 및 로그인

### 1. 회원가입
**[캡쳐 위치: 로그인 화면]**

1. 끄적끄적아지트 웹사이트에 접속합니다
2. "교사 로그인" 버튼을 클릭합니다
3. 이메일 주소와 비밀번호를 입력합니다
4. "회원가입" 버튼을 클릭하여 새 계정을 만듭니다

**필수 입력 정보:**
- 이메일 주소 (로그인 ID로 사용)
- 비밀번호 (안전한 비밀번호 설정 권장)

### 2. 로그인
**[캡쳐 위치: 교사 로그인 화면]**

1. 등록한 이메일과 비밀번호를 입력합니다
2. "로그인" 버튼을 클릭합니다
3. 최초 로그인 시 프로필 설정 화면으로 이동합니다

---

## 교사 프로필 설정

### 프로필 정보 입력
**[캡쳐 위치: 교사 프로필 설정 화면]**

최초 로그인 시 또는 프로필 수정 시 다음 정보를 입력합니다:

1. **이름**: 학생들에게 표시될 선생님 이름
2. **학교명**: 소속 학교 이름
3. **OpenAI API 키** (선택사항):
   - AI 자동 피드백 기능을 사용하려면 개인 API 키 입력
   - API 키가 없으면 시스템 기본 키 사용 (제한적)
   - API 키는 안전하게 서버에 암호화되어 저장됩니다

**💡 팁:**
- OpenAI API 키는 https://platform.openai.com 에서 발급받을 수 있습니다
- API 키 입력 시 학생 피드백에 더 많은 AI 기능을 활용할 수 있습니다

## 학급 생성 및 관리

### 1. 새 학급 만들기
**[캡쳐 위치: 학급 관리 화면]**

1. 대시보드에서 "학급 관리" 탭을 클릭합니다
2. "➕ 새 학급 만들기" 버튼을 클릭합니다
3. 학급 정보를 입력합니다:
   - **학급명**: 예) 3학년 1반, 국어 심화반

### 2. 학급 수정 및 삭제
**[캡쳐 위치: 학급 설정 메뉴]**

- 학급명은 언제든 수정 가능합니다
- 학급 삭제 시 모든 학생 데이터와 미션이 함께 삭제되므로 주의하세요
- 삭제 전 아카이브 기능으로 백업을 권장합니다
- 삭제된 학급은 3일 이내에 복구 가능합니다

## 학생 초대 및 관리

### 1. 학생 추가하기
**[캡쳐 위치: 학생 관리 화면]**

1. "학생 관리" 탭에서 "➕ 학생 추가" 버튼 클릭
2. 학생의 이름을 입력합니다
3. 시스템이 자동으로 **8자리 개인 코드**를 생성합니다
4. 생성된 코드를 해당 학생에게 전달합니다

**💡 중요:**
- 각 학생은 고유한 8자리 로그인 코드를 받습니다
- 학생은 이 코드로 로그인하여 학급에 참여합니다
- 코드는 학생 목록에서 언제든 확인 및 복사 가능합니다

### 2. 학생 정보 관리
**[캡쳐 위치: 학생 목록]**

학생 관리 화면에서 확인할 수 있는 정보:
- 학생 이름
- **8자리 로그인 코드** (복사 가능)
- 현재 레벨 및 포인트
- 제출한 글 개수
- 최근 활동 일시

### 3. 학생 제거
**[캡쳐 위치: 학생 관리 메뉴]**

- 학생 목록에서 개별 학생 제거 가능
- 제거된 학생의 데이터는 보관되며, 재추가 시 복구 가능

## 글쓰기 미션 만들기

### 1. 미션 기본 정보
**[캡쳐 위치: 미션 생성 화면 - 기본 정보]**

1. "미션 관리" 탭에서 "➕ 새 미션 만들기" 클릭
2. 기본 정보 입력:
   - **미션 제목**: 학생들에게 표시될 글쓰기 주제
   - **미션 종류**: 글쓰기 / 독서록 선택
   - **글의 장르**: 설명문, 논설문, 이야기글, 감상문 등
   - **가이드 메시지**: 학생들에게 주는 안내 문구

**예시:**
- 제목: "나의 꿈에 대해 써보기"
- 종류: 글쓰기
- 장르: 설명문
- 가이드: "여러분의 꿈과 그 꿈을 이루기 위해 노력하는 것을 자유롭게 써보세요"

### 2. 핵심 질문 설계
**[캡쳐 위치: 핵심 질문 설계 모달]**

학생들이 글을 쓰기 전에 생각을 정리할 수 있도록 돕는 질문을 설정합니다:

1. "🎯 핵심 질문 설계" 버튼 클릭
2. AI 자동 생성 또는 직접 입력 선택:
   - **AI 자동 생성**: 슬라이더로 1~5개 질문 개수 선택 후 "🪄 AI로 질문 자동 생성" 클릭
   - **직접 입력**: "➕ 질문 추가하기" 버튼으로 질문 추가

**예시 질문:**
- "나의 꿈은 무엇인가요?"
- "그 꿈을 갖게 된 계기는 무엇인가요?"
- "꿈을 이루기 위해 지금 하고 있는 노력은 무엇인가요?"

### 3. 미션 세부 설정
**[캡쳐 위치: 미션 세부 설정 섹션]**

**분량 및 설정:**
- **최소 글자수**: 학생이 작성해야 할 최소 글자수 (예: 200자)
- **문단 개수**: 최소 문단 개수 (예: 3문단)
- **댓글 허용**: 친구들이 댓글을 달 수 있는지 설정

**포인트 보상 설정:**
- **기본 포인트**: 글 제출 시 받는 기본 포인트
- **우수 글 추가 포인트**: 선생님이 우수 글로 선정 시 추가 포인트
- **댓글 작성 포인트**: 친구 글에 댓글 달 때 받는 포인트
- **댓글 받기 포인트**: 내 글에 댓글을 받을 때 받는 포인트

**💡 팁:**
- 포인트를 높게 설정하면 학생들의 참여도가 높아집니다
- 댓글 포인트를 활용하여 상호작용을 유도할 수 있습니다

### 4. 평가 루브릭 설정
**[캡쳐 위치: 평가 루브릭 섹션]**

학생 글을 평가할 기준을 설정합니다:

1. "평가 루브릭 설정" 섹션에서 평가 항목 추가
2. 각 항목별 설정:
   - **항목명**: 예) 내용의 충실성, 표현의 적절성
   - **배점**: 각 항목의 점수 (예: 10점, 20점)
   - **평가 기준**: 구체적인 평가 기준 설명

**기본 제공 항목:**
- 주제 이해도
- 내용의 충실성
- 표현의 적절성
- 맞춤법 및 문법

### 5. 미션 태그
**[캡쳐 위치: 태그 입력 섹션]**

- 미션을 분류하고 검색하기 쉽게 태그를 추가합니다
- 예: #꿈, #진로, #자기소개, #독서감상문

### 6. 미션 저장 및 발행
**[캡쳐 위치: 미션 저장 버튼]**

1. 모든 설정을 완료한 후 "💾 미션 저장" 버튼 클릭
2. 미션이 학생들에게 즉시 공개됩니다
3. 저장 후에도 수정 가능합니다

## 학생 제출물 확인 및 피드백

### 1. 제출물 목록 확인
**[캡쳐 위치: 미션 상세 - 제출 현황]**

1. "미션 관리" 탭에서 확인하고 싶은 미션 클릭
2. "제출 현황" 탭에서 학생별 제출 상태 확인:
   - ✅ 제출 완료
   - ⏳ 작성 중
   - ❌ 미제출

### 2. 학생 글 읽기
**[캡쳐 위치: 학생 제출물 상세 화면]**

1. 학생 이름을 클릭하여 제출물 확인
2. 확인할 수 있는 정보:
   - 학생이 작성한 글 전문
   - 핵심 질문에 대한 답변
   - 글자수, 문단 수 등 통계
   - 작성 일시 및 수정 이력

### 3. 피드백 작성
**[캡쳐 위치: 피드백 작성 모달]**

**방법 1: 직접 피드백 작성**
1. "✍️ 피드백 작성" 버튼 클릭
2. 피드백 내용 입력:
   - 잘한 점 칭찬하기
   - 개선할 점 안내하기
   - 격려 메시지 전달하기
3. "전송" 버튼으로 학생에게 피드백 전달

**방법 2: AI 피드백 활용**
1. "🤖 AI 피드백 생성" 버튼 클릭
2. AI가 자동으로 생성한 피드백 확인
3. 필요시 수정 후 전송

### 4. 평가 점수 입력
**[캡쳐 위치: 평가 루브릭 점수 입력]**

1. 설정한 평가 루브릭 항목별로 점수 입력
2. 총점이 자동으로 계산됩니다
3. 점수는 학생에게 공개되며 포인트로 변환됩니다

### 5. 우수 글 선정
**[캡쳐 위치: 우수 글 배지]**

1. 특별히 잘 쓴 글에 "⭐ 우수 글" 배지 부여
2. 우수 글로 선정된 글은:
   - 추가 포인트 획득
   - 학급 게시판에 하이라이트 표시
   - 다른 학생들의 모범 사례로 활용

## AI 자동 피드백 활용

### 1. AI 피드백 설정
**[캡쳐 위치: 교사 프로필 - API 키 설정]**

- 개인 OpenAI API 키를 등록하면 무제한 AI 피드백 사용 가능
- API 키가 없어도 시스템 기본 키로 제한적 사용 가능

### 2. AI 피드백 생성
**[캡쳐 위치: AI 피드백 생성 화면]**

**개별 피드백:**
1. 학생 제출물 화면에서 "🤖 AI 피드백 생성" 클릭
2. AI가 다음 내용을 자동 분석:
   - 주제 이해도
   - 내용의 논리성
   - 표현의 적절성
   - 맞춤법 및 문법 오류
   - 개선 제안

**일괄 피드백:**
1. 미션 관리 화면에서 "🤖 AI 일괄 피드백" 버튼 클릭
2. 여러 학생의 글에 대해 동시에 AI 피드백 생성
3. 진행 상황을 실시간으로 확인할 수 있습니다

### 3. AI 피드백 수정 및 전송
**[캡쳐 위치: AI 피드백 편집 화면]**

1. AI가 생성한 피드백을 검토합니다
2. 필요시 내용을 수정하거나 추가합니다
3. "전송" 버튼으로 학생에게 피드백 전달

**💡 팁:**
- AI 피드백은 기본 틀을 제공하므로, 개인화된 메시지를 추가하면 더 효과적입니다
- 학생의 성장 과정을 언급하면 동기부여에 도움이 됩니다

## 평가 루브릭 설정

### 1. 기본 루브릭 사용
**[캡쳐 위치: 평가 루브릭 기본 템플릿]**

시스템에서 제공하는 기본 평가 항목:
- 주제 이해도 (20점)
- 내용의 충실성 (30점)
- 표현의 적절성 (30점)
- 맞춤법 및 문법 (20점)

### 2. 커스텀 루브릭 만들기
**[캡쳐 위치: 루브릭 커스터마이징]**

1. 미션 생성 시 "평가 루브릭 설정" 섹션에서 항목 추가/수정
2. 각 항목별 설정:
   - 항목명
   - 배점
   - 평가 기준 상세 설명

**예시 - 독서 감상문 루브릭:**
- 책 내용 이해도 (25점)
- 자신의 생각 표현 (25점)
- 구체적인 예시 제시 (25점)
- 글의 구성 및 흐름 (25점)

### 3. 루브릭 재사용
**[캡쳐 위치: 루브릭 저장 및 불러오기]**

- 자주 사용하는 루브릭은 템플릿으로 저장 가능
- 새 미션 생성 시 저장된 템플릿 불러오기

## 게임 요소 관리

### 1. 포인트 시스템
**[캡쳐 위치: 게임 관리 화면]**

학생들이 포인트를 획득하는 방법:
- 글 제출: 미션별 설정 포인트
- 우수 글 선정: 추가 보너스 포인트
- 댓글 작성: 댓글당 포인트

## ❓ 자주 묻는 질문 (FAQ)

### 교사용 FAQ

**Q: API 키가 없으면 AI 피드백을 사용할 수 없나요?**
A: 아니요. 시스템 기본 키로 제한적으로 사용 가능합니다. 개인 API 키를 등록하면 무제한으로 사용할 수 있습니다.

**Q: 학급을 삭제하면 데이터가 완전히 사라지나요?**
A: 네, 학급 삭제 시 모든 데이터가 삭제됩니다. 삭제 전 아카이브 기능으로 백업하는 것을 권장합니다.

**Q: 학생이 부적절한 댓글을 달면 어떻게 하나요?**
A: 댓글 승인 시스템을 통해 부적절한 댓글을 차단할 수 있으며, 학생 관리 메뉴에서 경고를 줄 수 있습니다.

**Q: 미션을 수정하면 이미 제출한 학생에게 영향이 있나요?**
A: 기본 정보 수정은 영향이 없지만, 평가 루브릭 변경은 이미 평가한 점수에 영향을 줄 수 있으니 주의하세요.

**Q: 여러 학급을 동시에 관리할 수 있나요?**
A: 네, 제한 없이 여러 학급을 생성하고 관리할 수 있습니다.

### 학생용 FAQ

**Q: 글을 제출한 후에 수정할 수 있나요?**
A: 아니요, 제출 후에는 수정할 수 없습니다. 제출 전에 꼭 확인하세요.

**Q: 포인트를 잃을 수도 있나요?**
A: 일반적으로는 포인트가 줄어들지 않습니다. 단, 부적절한 행동 시 선생님이 포인트를 차감할 수 있습니다.

**Q: 친구 글에 댓글을 몇 개까지 달 수 있나요?**
A: 제한이 없습니다. 하지만 의미 있는 댓글을 다는 것이 중요합니다.

**Q: 용이 자라는 속도를 빠르게 할 수 있나요?**
A: 열심히 글을 쓰고 댓글을 달아 포인트를 많이 모으면 용이 빨리 자랍니다!

**Q: 배경을 구매하면 포인트가 사라지나요?**
A: 네, 배경 구매 시 해당 포인트가 차감됩니다. 신중하게 선택하세요.

**Q: 여러 학급에 동시에 참여할 수 있나요?**
A: 네, 여러 학급에 참여할 수 있으며 각 학급별로 다른 미션과 활동이 있습니다.
